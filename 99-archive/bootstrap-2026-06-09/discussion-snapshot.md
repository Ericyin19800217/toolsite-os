## 1. Hostinger 官方页面拆解

我们分析了 Hostinger 香港 Web Hosting 页面：

- 页面出售的不是单一虚拟主机，而是“AI 加持的一站式在线业务平台”。
- 核心能力包括主机、域名、SSL、WordPress、Node.js 应用、CDN、备份、邮箱、邮件营销、AI 建站、AI 应用生成、AI 客服/运维代理。
- 重要 AI 模块包括：
  - Kodee：AI 支持/运维代理。
  - Horizons：prompt-to-app / vibe coding 建站与应用生成器。
  - AI Website Builder / WordPress AI 插件。
  - Hostinger Reach AI 邮件营销。
  - hPanel / API / n8n 自动化底座。

关键判断：

```text
Hostinger 的强项是建站、部署、域名、主机、客服、控制面板一体化。
但从选题、SEO、推广、外链、复盘、长期维护来看，仍存在大量可自动化空间。
```
